
- NAMELIX TO-DO-LIST WEB APP (CRUD PROJECT)

Namelix name is system generated from online.

This project is one of my sample for CRUD project and practice project as a newbie Web Developer.
For this project, I search online on how to do basic CRUD project and found some references.
I am a self-taught programmer and still searching my way in this career path.  - vanprecilla

This project has login form, reset password form, and create user account. 

For the home page, it shows the to-do-list for the day and other section to pick to-do-list for another day. With hidden edit form to update the tasks.

Calendar option is for reference only.

With Account page menu to edit the user information and log-out form.

This project is purely code of html, css, javascript, php, and mysql with no framework.
